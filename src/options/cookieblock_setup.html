Thank you for installing CookieBlock!
CookieBlock automatically sorts cookies into different consent purposes. Please select which types of cookies you would like to prevent from being stored in your browser. This can be changed at any time by opening the extension settings page.

This category includes cookies that are typically required for essential services to operate. Deleting these may cause websites to break.


Functionality cookies usually serve to remember user preferences, such as language or website style, or enable non-essential functionality which can improve the user experience. Such cookies are usually persistent to keep settings stored across multiple browsing sessions.


Analytics cookies serve to collect statistical information about how visitors use the current website. This is generally used to measure the website performance, and to improve the service. Measurements include page visits, bounce rate, dwell time and more. Under normal circumstances, the collected data should not contain sensitive information that could identify the user.


Advertising cookies track visitors across multiple websites and collect sensitive, personally identifiable information. The goal is to display optimized advertisements that best match the personal interests of the users, or to sell the data that has been collected to other interested parties.

Debug mode will disable cookie removal and enable debug output in the browser console. Only used for testing purposes.
